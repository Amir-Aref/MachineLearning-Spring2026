## Submission Instructions

Submission instructions will be announced by the course staff.

Unless otherwise stated:

- Submit your work before the deadline.
- Follow the required file naming format.
- Include all requested files.
- Make sure your code runs without errors.
- Do not submit unnecessary large files, temporary files, or cache folders.
- Keep a backup of your submitted work.
